# I Have Nothing But Questions

Gallery shooter game for CMPM 120.

## Description

In this game, the player controls a monkey head at the bottom of the screen. Question marks and exclamation marks fall from the top, and the player throws bananas upward to destroy them.

The game is about having too many questions coming at you at once.

## Controls

- A / D or Arrow Keys: Move
- Space: Throw banana / Start game
- R: Restart after game over
- T: Return to title screen
- C: Credits screen
